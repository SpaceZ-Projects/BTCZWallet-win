PayGate API Methods & Examples

Method: help

Returns a list of available methods.

http://{{ host }}:{{ port }}/api/?method=help

Method: Invoice Example

Example to view an invoice with a QR code:

http://{{ host }}:{{ port }}/invoice/f22c44cb-e26a-4022-864f-00f0d523d48a
Copied!

Method: check_payment

Check the status of a payment using the invoice ID:

Example:

http://{{ host }}:{{ port }}/api/check_payment/f22c44cb-e26a-4022-864f-00f0d523d48a
